About Dyslexia

Dyslexia is a learning disorder that affects reading and related language-based processing skills. It can make it difficult for people to read, write, and spell. However, with the right tools and support, people with dyslexia can excel in many areas.

Common challenges include:

  • Difficulty with reading fluency
  • Problems with word recognition
  • Issues with spelling and writing
  • Challenges with phonological processing

How This Extension Helps

The Dyslexia Helper extension provides various tools to make web content more accessible:

  • Font Changes: Uses dyslexia-friendly fonts like OpenDyslexic
  • Text Spacing: Adjusts letter, word, and line spacing
  • Color Overlays: Adds colored overlays to reduce visual stress
  • High Contrast: Increases contrast for better readability
  • Text-to-Speech: Reads selected text aloud
